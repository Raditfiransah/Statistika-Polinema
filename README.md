# Statistika-Polinema

## Deskripsi

Repositori ini berisi materi dan tugas-tugas perkuliahan Statistika Komputasi untuk program studi Teknik Informatika, Politeknik Negeri Malang (Polinema).

## Struktur Repositori

```
Statistika-Polinema/
├── Materi/
│   ├── Week 1 - Pengenalan Statistika.pptx.pdf
│   ├── Week 2 - Penyajian Data.pdf
│   ├── Week 3a - Pemusatan Data.pdf
│   └── Week 3b - Penyebaran Data_update.pdf
├── Tugas-1/
│   ├── data/
│   │   └── data.csv
│   └── notebook/
│       └── Jawaban.ipynb
├── Tugas-2/
│   └── notebook/
│       └── Jawaban.ipynb
├── .gitignore
└── README.md
``` Instalasi

1. Clone repositori:

   ```bash
   git clone <url-repositori>
   cd Statistika-Polinema
   ```

2. Buat dan aktifkan virtual environment:

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Install dependensi:
   ```bash
   pip install pandas numpy matplotlib plotly
   ```

## Penggunaan

Buka notebook menggunakan Jupyter Notebook atau JupyterLab:

```bash
jupyter notebook
```
